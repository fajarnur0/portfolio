Web Steganography For Message

Website steganografi menggunakan metode PVD (Pixel Differencing Value) merupakan website untuk menyembunyikan pesan rahasia ke dalam media citra digital secara tak kasat mata. Dibuat menggunakan HTML, Tailwind CSS, dan Javascript.

Web Enkripsi Pesan Metode RC4

Website enkripsi pesan menggunakan metode RC4 (Rivest Cipher 4) merupakan website untuk melakukan pengacakan pesan rahasia menjadi karakter-karakter ASCII sesuai dengan algoritma RC4, sehingga pesan tidak dapat dikenali dan terlindungi. Dibuat menggunakan HTML, Tailwind CSS, dan Javascript.
